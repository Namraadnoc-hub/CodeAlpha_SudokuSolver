# CodeAlpha Sudoku Solver

## Overview

The CodeAlpha Sudoku Solver is a C++ based application that solves Sudoku puzzles using the Backtracking Algorithm.

The project focuses on recursion, problem solving, and algorithm implementation.

---

## Features

* Automatic Sudoku solving
* 9x9 grid representation
* Row validation
* Column validation
* 3x3 sub-grid checking
* Recursive backtracking algorithm

---

## Technologies Used

* C++
* Arrays
* Recursion
* Backtracking Algorithm

---

## Algorithm Used

### Backtracking

The algorithm:

1. Finds an empty cell.
2. Tries possible numbers from 1-9.
3. Checks whether the number is valid.
4. Recursively continues solving.
5. Backtracks when a solution path fails.
